# Semantic-Segmentation
我将会将许多的语义分割的模型上传到这个仓库中提供给大家学习使用，包括segnet,pspnet,unet,deeplab_v3_plus等。  

# 用法
你可以下载后进入你所想要训练的模型的文件夹，然后运行train.py进行训练。  
在训练之前，需要先下载数据集，并将其存储到dataset中。  
大家关心的多分类的代码在Muiti_Class_deeplab_Mobile里。
